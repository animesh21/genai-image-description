# Python project to serve as a backend for Gen AI based image description generator

## Steps to run
1. Clone this repository using command `git clone git@github.com:animesh21/genai-image-description.git`
2. Get into the project directory using command `cd genai-image-description`
3. Run `uv sync` to install the dependencies
4. Run `uv run main.py`
5. Go to http://127.0.0.1:8000/health to check if the app is up and running
6. Once the app is up and running, go to http://127.0.0.1:8000/docs to check the API docs and use accordingly.
